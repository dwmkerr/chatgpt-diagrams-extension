  y: number;
};

console.log(vector.x); // Output: 3
console.log(vector.y); // Output: 5

In this example, the Vector2D type represents a 2D vector. It consists of two properties: x and y, both of which are numbers. You can create instances of the Vector2D type by assigning values to the x and y properties.
